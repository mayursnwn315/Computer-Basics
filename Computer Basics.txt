1.What is a computer?


Answer: A computer is an electronic device that processes data, performs calculations, and executes predefined instructions to perform various tasks. It consists of hardware components like the central processing unit (CPU), memory, storage, and input/output devices.


2.What is RAM?


Answer: RAM (Random Access Memory) is a type of computer memory that is used to store data and machine code currently being used and processed by the CPU. It allows for quick read and write operations, providing fast access to data during the computer's operation.


3.Where is data stored in a computer?


Answer: Data in a computer is stored in various types of storage devices. This includes RAM for temporary storage during active processing and long-term storage devices such as hard drives, solid-state drives (SSDs), or other secondary storage media.


4.What is the input device used to type text and numbers on a document in the computer system?


Answer: The input device used to type text and numbers on a document in the computer system is a keyboard.


5.What are the output devices?


Answer: Output devices are hardware components that provide information or results from a computer. Examples include monitors (display screens), printers, speakers, and projectors.


6.Which is the input device that allows a user to move the cursor or pointer on the screen?


Answer: The input device that allows a user to move the cursor or pointer on the screen is a mouse.


7.Which language is directly understood by the computer without a translation program?


Answer: Machine language or binary code is directly understood by the computer without the need for a translation program. It consists of 0s and 1s that represent machine instructions.


8.What are Input Devices?


Answer: Input devices are hardware components that allow users to provide data and instructions to a computer. Examples include keyboards, mice, touchscreens, scanners, and microphones.
